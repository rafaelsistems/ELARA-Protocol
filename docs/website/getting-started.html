Installation

Add ELARA crates to your Cargo.toml:

[dependencies] elara-core = "0.2.0" elara-crypto = "0.2.0" elara-time = "0.2.0" elara-state = "0.2.0" elara-wire = "0.2.0" elara-transport = "0.2.0"

Or use cargo add:

cargo add elara-core elara-crypto elara-time elara-state elara-wire elara-transport

Quick Start

Here's a minimal example to get you started:

use elara_core::{NodeId, Event, EventType}; use elara_crypto::Identity; use elara_time::TimeEngine; use elara_state::StateField; #[tokio::main] async fn main() -> Result<(), Box> { // Generate a cryptographic identity let identity = Identity::generate(); let node_id = identity.node_id(); println!("ELARA Node ID: {}", node_id); // Create time engine for temporal coordination let mut time_engine = TimeEngine::new(node_id); // Create state field for distributed state let mut state_field = StateField::new(node_id); // Create an event with temporal awareness let event = Event::new( EventType::PresenceUpdate, b"Hello, ELARA!".to_vec(), time_engine.perceptual_time() ); println!("Created event at perceptual time: {:?}", event.timestamp()); Ok(()) }
Success! You've created your first ELARA node with identity, time awareness, and event handling.

Core Concepts

🔐 Identity

Every ELARA node has a cryptographic identity:

use elara_crypto::Identity; // Generate new identity let identity = Identity::generate(); let node_id = identity.node_id(); // Export for persistence let exported = identity.export()?; // Import from storage let identity = Identity::import(&exported)?;

⏱️ Dual Clock System

ELARA uses two time systems:

use elara_time::{TimeEngine, PerceptualClock, StateClock}; let mut engine = TimeEngine::new(node_id); // Perceptual time: smooth, for UX let perceptual = engine.perceptual_time(); // State time: correctable, for consensus let state = engine.state_time(); // Update based on network observations engine.update_peer_time(peer_id, remote_time, rtt)?;

🔄 State Field

Distributed state as a physical field:

use elara_state::{StateField, StateAtom, Authority}; use elara_core::StateId; let mut field = StateField::new(node_id); // Create state atom let atom = StateAtom::new( StateId::generate(), Authority::Open, data, version_vector ); // Apply to local field field.apply_atom(atom)?;

📡 Wire Protocol

Binary encoding with packet classes:

use elara_wire::{Frame, FrameBuilder, PacketClass}; use elara_crypto::SecureFrameProcessor; let frame = FrameBuilder::new() .with_class(PacketClass::Perceptual) .with_payload(data) .build()?; // Encrypt for transport let secure_frame = crypto.encrypt(frame, session_key)?;

Examples

Simple Chat Application

use elara_core::{Event, EventType, MessageId}; use elara_state::StateField; // Create text message event let message_event = Event::new( EventType::TextMessage, b"Hello from ELARA!".to_vec(), time_engine.perceptual_time() ); // Store in state field let message_atom = StateAtom::from_event(message_event, Authority::Open); state_field.apply_atom(message_atom)?;

Network Time Synchronization

use elara_time::{NetworkQuality, HorizonConfig}; // Configure horizons based on network quality let quality = NetworkQuality::from_rtt_stats(rtt_samples); let config = HorizonConfig::from_quality(quality); time_engine.set_horizons(config); // Time now adapts to network conditions let adaptive_time = time_engine.adaptive_time();

State Reconciliation

use elara_state::ReconciliationEngine; let mut reconciler = ReconciliationEngine::new(); // Merge remote state let result = reconciler.reconcile(&local_field, &remote_field)?; // Check convergence if result.is_converged() { println!("State converged!"); } else { println!("Divergence detected: {:?}", result.divergence()); }
